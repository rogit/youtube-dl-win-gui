# youtube-dl-win-gui
Windows (7,8,10) GUI of the popular [youtube-dl](https://rg3.github.io/youtube-dl/) media downloader. [Supported sites](https://rg3.github.io/youtube-dl/supportedsites.html)

## Downloads
* [youtube-dl-win-gui.2020.06.02.zip](https://github.com/rogit/youtube-dl-win-gui/blob/master/builds/youtube-dl-win-gui.2020.06.02.zip) 10.1 MiB

## Installation

1. Download & extract the source
2. Run youtube-dl-win-gui.exe

## Requirements

1. [youtube-dl] (https://ytdl-org.github.io/youtube-dl/download.html)
Build includes version 2020.01.24

2. [PHP] (https://windows.php.net/download/)
Build includes version 7.4.2

3. [windows-desktop-web-app] (https://github.com/rogit/windows-desktop-web-app)

4. Microsoft Visual C++ 2010 Redistributable Package (https://www.microsoft.com/en-US/download/details.aspx?id=5555)
Required for youtube-dl. Build does not include it.

5. Microsoft Visual C++ Redistributable for Visual Studio 2015, 2017 and 2019 (https://support.microsoft.com/en-us/help/2977003/the-latest-supported-visual-c-downloads)
Required for PHP.  Build does not include it.
