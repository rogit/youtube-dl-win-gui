# youtube-dl-win-gui
Windows (7,8,10) GUI of the popular [youtube-dl](https://youtube-dl.org/) media downloader.

## Screenshot
![youtube-dl-win-gui main window](https://raw.githubusercontent.com/rogit/youtube-dl-win-gui/master/builds/screenshot.png)

## Download
[youtube-dl-win-gui.2022.06.25.zip](https://github.com/rogit/youtube-dl-win-gui/raw/master/builds/youtube-dl-win-gui.2022.06.25.zip) 11 MiB

## Installation

1. Download & extract the .zip archive
2. Run youtube-dl-win-gui.exe

## Requirements

1. Modern web-browser set as default. Not Internet Explorer.

2. FFmpeg (https://www.ffmpeg.org/download.html#build-windows). For creating mp3.
